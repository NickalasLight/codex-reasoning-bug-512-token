## Credit Policy

- Credit public contributors by GitHub handle and preferred display name when available.
- Link each credit to a PR, issue, commit, or evidence artifact.
- Distinguish contributor-reported evidence from maintainer-verified reproduction.
- Do not publish private emails, raw transcripts, private paths, prompts, tool outputs, credentials, customer/user data, or non-public repository details.
- Prefer named evidence folders under `evidence/external/<contributor>-<topic>-<date>/` for substantial external analyses.
